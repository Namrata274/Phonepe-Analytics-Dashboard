# Phonepe-Analytics-Dashboard
# 1. Project Title 
💸 PhonePe Pulse Analytics: India’s Digital Payment Insights Dashboard                                                     
A dynamic, interactive data visualization tool built to explore PhonePe transaction trends across India—focusing on regional comparisons, payment categories, growth patterns, and user adoption.

# 2. Short Description / Purpose 
The PhonePe Analytics Dashboard is a visually engaging and interactive Power BI report designed to help users explore and analyze India’s digital payment ecosystem. The dashboard highlights transaction trends, service‑wise values, user demographics, and behavioral insights—empowering analysts, fintech professionals, and strategists to understand adoption patterns and growth opportunities in the digital payments space. 

# 3. Tech Stack  
List the key technologies used to build the dashboard.  

Example: The dashboard was built using the following tools and technologies:                              
• 📊 Power BI Desktop – Main data visualization platform used for report creation.  
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.  
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.  
• 📝 Data Modeling – Relationships established among tables (Date, Transaction, Users) to enable cross-filtering and aggregation.  
• 📁 File Format – .pbix for development and .jpeg for dashboard previews.    

# 4. Data Source
PhonePe Pulse (2024) & curated sample transaction datasets.   
The dataset is structured into multiple tables covering time‑series trends, service‑wise values, and user attributes, enabling clear analysis of how people used PhonePe in 2024 across regions, services, and age groups.   

# 5. Features / Highlights  

**Business Problem**  
Digital payments in India are growing very fast, but it is hard for businesses, analysts, and policymakers to clearly see transaction trends, service categories, and user behavior.  
Important questions like: Which service has the highest transaction value? How do people use PhonePe on weekdays vs weekends?
Which age group is using it the most? … are not easy to answer with raw data.  

• Goal of the Dashboard To provide an interactive and simple visual tool that:
Shows PhonePe transactions across India.
Helps in making better decisions for fintech strategy, marketing, and customer engagement.  
Finds patterns in services, user demographics, and payment behavior.  

**Walkthrough of Key Visuals**  
**• Key KPIs (Top Section):** Total Transactions (~300K), Total Value (~₹3.47bn), Unique Users (~108K), Success Rate (~95.93%).  
**• Transactions Over Time (Line Chart):** Monthly trend of transaction counts and values in 2024.  
**• Service Transaction Value (Bar Chart):** Loans (~2.5bn), Insurance (~0.5bn), Transfers (~0.4bn), Recharge (~0.1bn).  
**• Top 5 Users (Bar Chart):** Shows the users with the highest transaction values.  
**• Weekday vs Weekend Usage (Pie Chart):** Weekday (~71.6%), Weekend (~28.4%).  
**• Age Segment Contribution (Donut Chart):** Gen X (~35%), Millennials (~35%), Gen Z (~21%), Others (~9%).  

**Business Impact & Insights**  
  **• Service Focus:** Loans give the highest value, so fintech companies can focus more on lending products.  
  **• User Behavior:** More payments happen on weekdays, showing strong work‑related usage.  
  **• Demographics:** Gen X and Millennials are the biggest contributors, useful for targeted marketing.  
  **• Growth Areas:** Insurance and transfers have scope to grow with better promotions.

# 🛠️ Tooltips  
# 1. Transactions Over Time Tooltip
Shows extra details for each month, including total transaction value and number of transactions. This helps users quickly compare monthly trends without opening another chart.  
# 2. Service Transaction Value Tooltip
Displays detailed values for each service category like Loans, Insurance, Transfers, Recharge. Users can hover to see exact numbers and understand which service contributes the most.  
# 3. Age Segment Contribution Tooltip
Explains the share of different age groups (Gen X, Millennials, Gen Z, Others) in total transactions. Hovering shows clear percentages, making it easy to see which age group is most active.
